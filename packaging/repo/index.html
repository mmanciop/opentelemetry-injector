OpenTelemetry Auto-Instrumentation

Current version: @@VERSION@@

Linux packages for automatic instrumentation of Java, Node.js, and .NET applications using OpenTelemetry.

Quick Install (Debian/Ubuntu)

# Add the repository
echo "deb [trusted=yes] @@REPO_URL@@/debian stable main" | sudo tee /etc/apt/sources.list.d/opentelemetry.list

# Update and install
sudo apt update
sudo apt install opentelemetry

Quick Install (RHEL/Fedora/Amazon Linux)

# Add the repository
sudo curl -o /etc/yum.repos.d/opentelemetry.repo @@REPO_URL@@/rpm/opentelemetry-injector.repo

# Install
sudo dnf install opentelemetry

Available Packages

  • opentelemetry - Metapackage that installs all components
  • opentelemetry-injector - LD_PRELOAD-based auto-instrumentation injector
  • opentelemetry-java-autoinstrumentation - Java agent for automatic instrumentation
  • opentelemetry-nodejs-autoinstrumentation - Node.js agent for automatic instrumentation
  • opentelemetry-dotnet-autoinstrumentation - .NET agent for automatic instrumentation

Individual Package Installation

You can install individual packages if you only need specific language support:

# Install only Java instrumentation
sudo apt install opentelemetry-injector opentelemetry-java-autoinstrumentation

# Install only Node.js instrumentation
sudo apt install opentelemetry-injector opentelemetry-nodejs-autoinstrumentation

# Install only .NET instrumentation
sudo apt install opentelemetry-injector opentelemetry-dotnet-autoinstrumentation
